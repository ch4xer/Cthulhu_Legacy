# 克苏鲁的遗产

本项目旨在搜集经典的克苏鲁原文作品，并提供由大语言模型(Deepseek)生成的中文翻译，方便整理和阅读。

本项目搜集的文章列表主要来自于国内引进的克苏鲁神话相关书籍目录，他们筛选出了很多优秀的作品，然而我一直不太满意其翻译质量，因此决定自己动手使用大语言模型进行翻译，力图在保留原文风格的同时，尽量做到通顺易读。

> **注意**：本项目仅供学习和研究使用，不得用于商业用途。
